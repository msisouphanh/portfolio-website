## E-Portfolio

A personal e-portfolio website designed to showcase my projects, skills, and experience.

## Technologies
- `React`
- `TypesScript`
- `HTML`
- `Tailwind CSS`
- `Vite`

## Running the Project

1. Clone the repository to your local machine.
2. Install Vite by running: `npm install vite@latest`
3. Install React Router with: `npm install react-router-dom`
4. Start the development server using: `npm run dev`
5. Open your browser and navigate to `http://localhost:5183`
